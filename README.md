# Photoshop Script: Sort Selected Group (Folders Only)

## Overview
This script sorts **only the currently selected folder (group)** in Adobe Photoshop.  
It will alphabetically and numerically sort **groups inside that folder**, including any nested sub-groups.

It does **not** affect individual layers.

---

## Requirements
- Adobe Photoshop 2022 or newer (tested up to Photoshop 2026)
- ExtendScript (.jsx) support enabled

---

## How to Use

### 1. Select a Folder (Group)
In the **Layers panel**, click to select the folder you want to sort.

> ⚠️ Important:  
> You must select a **group/folder**, not a normal layer.

---

### 2. Install the Script

#### Option A — Temporary Run
1. Open Photoshop
2. Go to:
3. Select the `.jsx` script file
4. Click **Open**
